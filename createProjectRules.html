Tребования к проекту

PolessCrowdFinding публикует проекты любой тематики и направленности. Основные требования к проектам:

Качественное оформление
Текст описания должен быть написан грамотным языком и четко структурирован. Также обязательным является наличие качественных фотоматериалов, иллюстраций или инфографики. Желательно наличие видео-обращения.
Актуальные подарки
Список подарков для спонсоров должен быть тщательно продуман и напрямую связан с результатами проекта. У проектов с отстраненными подарками нет шансов на успех.
Обоснованный бюджет
Бюджет проекта не должен быть раздут. Он должен включать в себя тот минимум, который необходим для успешной реализации проекта.
Соответствие закону и правилам
Проект не должен нарушать законов РБ и должен соответствовать правилам платформы PolessCrowdFinding.

Требования к автору

На данный момент PolessCrowdFinding работает исключительно с физическими лицами. Для создания проекта автор должен соответствовать ряду требований:

Возраст: от 18 лет.
Гражданин Республики Беларусь.
Имеется расчетный счет в любом белорусском банке.

Процесс создания проекта

Создание проекта состоит из нескольких последовательных этапов:

Оформление проекта с помощью «редактора проектов».
Модерация оформленного проекта.
Подписание с автором договоров на публикацию проекта на платформе и сбор средств.
Публикация проекта на сайте.
